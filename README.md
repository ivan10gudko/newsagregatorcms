# ️ News Aggregator - Sanity CMS

This repository contains the Headless CMS (content backend) for the News Aggregator application, built with Sanity Studio.

**[Go to Frontend Repository](https://github.com/ivan10gudko/news_agregator.git)**

##  Live Studio
**Admin Panel:** https://www.sanity.io/@oVwiQ4lOG/studio

*(Note: The studio is private. Reviewers can log in with GitHub/Google and click "Request Access". I will approve it immediately!)*

## Features
- **Dynamic Configuration:** Manages topics (categories) and news sources for the frontend application.
- **Custom Validation:** Includes asynchronous validators to prevent duplicate sources and empty keywords.

## Tech Stack
- Sanity Studio v3
- React
- TypeScript

## How to Run Locally

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the local studio:**
    ```bash
    npm run dev
    ```
